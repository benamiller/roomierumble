Tournament Format

Ludwig is not all there. Accurate format to come.

League of Legends

Detailed format for League of Legends. Each match will be a Best of 3. Points awarded for series wins. Specific map pick/ban rules apply.

  • Game Mode: 5v5 Summoner's Rift, Tournament Draft
  • Win Condition: Destroy enemy Nexus
  • Points: 3 for win, 1 for 1-2 loss, 0 for 0-2 loss

NBA 2K

NBA 2K will be played in a round-robin format. Each team plays each other once. Standard game settings, 5 minute quarters.

  • Game Mode: MyTEAM Friendly Match
  • Points: 2 for win, 0 for loss
  • Tie Breakers: Head-to-head, then point differential

CS2

CS2 matches are Best of 1 on a pre-determined map pool. MR12 format. Teams will play in a group stage followed by playoffs.

  • Game Mode: Competitive 5v5, MR12
  • Map Pool: To be announced
  • Points: Based on group stage ranking

Trackmania

Trackmania will feature a series of cup mode races on custom maps. Points awarded based on final ranking in each cup.

  • Game Mode: Cup Mode (Points Limit)
  • Tracks: 5 custom tracks per week
  • Points: Top 8 players per race score points (10, 8, 6, 5, 4, 3, 2, 1)
